Allow customers to contact you using Line Messenger

== Description ==

Let's make it easier for your customers to contact you! Probably some of your users are using Line Messenger, so it's time to give them the option to contact you by using Line Messenger.
With «Smartarget Line - Contact Us» you can add a small icon on the bottom-right corner with your phone number and short message as «Contact us» or «message us» etc. When the user clicks on the icon - a new tab will pop with Line Messenger and chat window with you.

Line Chat Contact Us plugin works also with WooCommerce, Elementor and Oxygen Builder.

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
